下载主题

https://gitee.com/Teily/hexo-theme-melody

部署验证

下载完成后，先测试主题是否OK

npm install hexo-server hexo-browsersync hexo-renderer-jade hexo-renderer-stylus hexo-generator-search --save-dev
hexo g
hexo s

部署后，通过访问http://localhost:4000/来确定是否成功
